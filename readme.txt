== Description ==

Display and follow the conversation on social media (Twitter, Instagram and many more) about a #hashtag on your site.

Watched social net is a plugin that allows you to put together all your social media content (including #hashtags, 
@accounts, videos, photos, and tweets) in an amazing social wall on your WordPress site. Watched will automatically 
display the post via hashtag on screens of any size. We support hashtag conversations from Twitter & Instagram.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/WatcherSocialNet` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Watcher Social Net Name screen to configure the plugin
4. You must install Curl in your server
